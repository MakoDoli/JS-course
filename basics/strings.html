We shall not use 'new String()' or 'new Number()' constructor functions to create/asign string/number variables

String.search() method takes regex as argument, while String.indexOf() takes literal substring as argument

String.indexOf() returns position of first occurrence of literal substring argument

String.search() returns position of first occurrence of regex argument

String.match() returns array of all occurrences of regex argument

.length is property, not method
