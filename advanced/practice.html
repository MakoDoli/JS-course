Js engine takes the code and performs three steps:

  • Parsing
  • Compilation
  • Execution

PARSING

During parsing code is converted to tokens (each token for each syntax piece), then syntax parser builds AST - Abstract Syntax Tree. AST is a JSON object with key-values describing start, end, body, type of code as well as other details

COMPILATION

Interpretter

Interpretter reads code line by line. Interpretter is fast

Complier reads whole code and modifies it to more optimized code. Complier is efficient

JS uses JIT (just in time) complilation -

AST goes to Interpretter and interpreter converts high-level code to byte code and compiler tries to optimize the code and then code is sent to execution
