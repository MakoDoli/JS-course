Every object has .contructor property, which references to function that was used to create this object

Factory functions are simple functions which take parameters and create and return objects using parameters as object key values

Contructor function should be written with capital Letter, it creates pack of properties and methods of future object. The object is actually created by keyword 'new' which creates empty object inside the contructor function, assignes this empty object to 'this' variable, drops bunch of properties and methods in object and returns newly baked object. all that behind scenes
