js has only one type of numbers (whole numbers, decimals - all are numbers)

e - exponentiation - 10 times number after e. 2e3 = 2000, and with "-" 2e-3 = 0.0002

There are two main types of data in javascript - primitives and objects

METHOD DOES something, PROPERTY IS something

Javascript has built-in objects - global objects -date, math

Primitives - numbers, strings and booleans - arent objects, but have built-in object counterpart, so numbers have access to the object with N (Number)

numbers can not access built-in String object and its methods, as well as strings can not access built-in Number object and its methods
